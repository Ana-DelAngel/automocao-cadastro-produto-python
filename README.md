# automocao-cadastro-produto-python
📌 Sobre o projeto

Projeto desenvolvido para automatizar o cadastro de produtos em um sistema web utilizando Python. Os dados são lidos de uma planilha CSV e preenchidos automaticamente na aplicação.

🚀 Funcionalidades
Abertura automática do navegador
Login no sistema
Leitura de dados de uma planilha CSV
Cadastro automático dos produtos
🛠️ Tecnologias utilizadas
Python
Pandas
PyAutoGUI
📚 Conceitos praticados
Automação de tarefas
Manipulação de arquivos CSV
Utilização de bibliotecas Python
Estruturas de repetição
Organização do código
🎯 Objetivo

Reduzir o tempo gasto em tarefas repetitivas e praticar automação de processos utilizando Python.
